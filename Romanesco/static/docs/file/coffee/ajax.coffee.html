    load an area from the server the project coordinate system is divided into square cells of size g.scale an Area is an object { pos: Point, planet: Point } corresponding to a cell (pos is the top left corner of the cell, the server consider the cells to be 1 unit wide (1000 pixels)) a load does:

    • build a list of Area overlapping area and not already loaded
    • define a load limit rectangle equels to area expanded to 2 x g.scale
    • remove RItems which are not within this limit anymore AND in an area which must be unloaded (do not remove items on an area which is not unloaded, otherwise they wont be reloaded if user comes back on it)
