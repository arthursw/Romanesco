~ (void) addItem(name, parameter, item, datFolder, resetValues)

doctodo: copy deault parameter doc here add a controller to dat.gui (corresponding to a parameter, from a tool or an item)

Parameters:

  • name ( String ) — of the parameter (short name without spaces, same as RItem.data[name] )
  • the ( Parameter ) — parameter to add
  • optional ( RItem ) — RItem, the controller will be initialized with item.data if any
  • folder ( DatFolder ) — in which to add the controller
  • (optional) ( resetValues ) — reset value to default (create a new default if parameter has a defaultFunction)

~ (void) updateParameters(tools, resetValues = false)

update parameters according to the selected tool or items

Parameters:

  • or ( { tool: RTool constructor, item: RItem ) — Array of { tool: RTool constructor, item: RItem }] list of tools from which controllers will be created or updated
  • reset ( Boolean ) — controller values or let them untouched (values must be reset when selecting a new tool, but not when creating another similar shape... this must be improved)
