Class: ScreenshotTool

Defined in: coffee/tools.coffee
Inherits: RTool

Overview

todo: ZeroClipboard.destroy() ScreenshotTool to take a screenshot and save it or publish it on different social platforms (facebook, pinterest or twitter)

  • the user will create a selection rectangle with the mouse
  • when the user release the mouse, a special (temporary) resizable RDiv (RSelectionRectangle) is created so that the user can adjust the screenshot box to fit his needs (this must be imporved, with better visibility and the possibility to better snap the box to the grid)
  • once the user adjusted the box, he can take the screenshot by clicking the "Take screenshot" button at the center of the RSelectionRectangle
  • a modal window asks the user how to exploit the newly created image (copy it, save it, or publish it on facebook, twitter or pinterest)

Instance Method Summary

  • # (void) getDescription() Get description input value, or default description: "Artwork made in Romanesco"
  • # (void) begin(event) create selection rectangle
  • # (void) update(event) update selection rectangle
  • # (void) end(event)
      remove selection rectangle
    • return if rectangle is too small
    • create the RSelectionRectangle (so that the user can adjust the screenshot box to fit his needs)
  • # (void) extractImage() Bound Extract image and initialize & display modal (so that the user can choose what to do with it) todo: use something like rasterizeHTML.js to render RDivs in the image
  • # (void) saveImage(callback) Some actions require to upload the image on the server makes an ajax request to save the image
  • # (void) publishOnFacebook() Bound Save image and call publish on facebook callback
  • # (void) publishOnFacebook_callback(result) Bound (Called once the image is uploaded) add a facebook dialog box in which user can add more info and publish the image todo: check if upload was successful?
  • # (void) publishOnFacebookAsPhoto() Bound
      log in to facebook (if not already logged in)
    • save image to publish photo when/if logged in
  • # (void) publishOnFacebookAsPhoto_callback(result) Bound (Called once the image is uploaded) directly publish the image todo: check if upload was successful?
  • # (void) publishOnPinterest() Bound Save image and call publish on pinterest callback
  • # (void) publishOnPinterest_callback(result) Bound (Called once the image is uploaded) add a modal dialog to publish the image on pinterest (the pinterest button must link to an image already existing on the server) todo: check if upload was successful?
  • # (void) downloadPNG() Bound on download png button click: simulate a click on the image link (chrome will open the save image dialog, other browsers will open the image in a new window/tab for the user to be able to save it)
  • # (void) downloadSVG() Bound on download svg button click: extract svg from the paper project (in the selected rectangle) and click on resulting svg image link (chrome will open the save image dialog, other browsers will open the image in a new window/tab for the user to be able to save it)
  • # (void) copyURL() Bound nothing to do here: ZeroClipboard handles it
