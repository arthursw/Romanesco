Constructor Details

# (void) constructor(urlList, onLoadCallback)

The first url will be loaded, and taken as source if the load succeeds. Otherwise the second url will be loaded and so on until a file succeeds or all urls have been tested.

Parameters:

  • the ( Array of string ) — list of urls of the files to load. (one sound file, many formats: list of urls to each format of the same file)
  • called ( function ) — after the file is loaded
